# 🛠️ CLD Exercise 5: Reading and Writing Data from a Configuration File

## 🎯 重要概念 (Core Concepts)
讀寫設定檔 (.ini)。重點在於動態讀取數值、布林與字串叢集，並將資料寫入指定的二維陣列 (2 row by 4 column) 索引位置。

---

## ⚠️ 易錯內容 (Gotchas / Common Mistakes)
> [!WARNING]
> 陣列的「零基索引 (zero-based indexing)」概念，容易在計算行與列時發生「差一錯誤 (Off-by-one error)」。資料型態與配置檔的對應也必須確保一致。
