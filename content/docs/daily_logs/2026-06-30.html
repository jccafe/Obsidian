🔍 錯誤與學習 (Errors & Learning)


1. Powershell 轉義與 Python 單行指令語法錯誤：
  • *現象*：在控制台直接執行含有大括號、引號與換行符的 python -c 命令時，Powershell 的解析機制極易觸發 SyntaxError: unterminated string literal。

  • *解決*：將複雜操作（如更新 Index 連結、搬移檔案）改為在 scratch/ 下撰寫獨立的 Python 檔案，再以 python .py 執行，語法更清晰且不易出錯。

2. 000_Index 內文 replace 取代失敗：
  • *現象*：在 create_cld_guide_note.py 中更新首頁時，replace 邏輯由於對前導空格與反斜線匹配不精準而失效。

  • *解決*：直接對檔案內容進行特徵匹配（相容換行與空格符），並改寫成精準替換，隨後以 Python 腳本成功覆寫。

3. Google 憑證與 Puppeteer 鎖定權限衝突：
  • *現象*：試圖將 notebooklm-mcp.jc.tsjh 重命名覆寫為 notebooklm-mcp 時，由於 Puppeteer 進程運行中佔用了部分 chrome_profile 檔案，引發 PermissionError: [WinError 5] 存取被拒。

  • *解決*：使用 Python 腳本避開鎖定檔並覆寫了 library.json 等配置，但由於雲端共用權限與 session cookies 差異，最終靈活轉向直接從本地 artifacts 中提取出 4 篇考題文字並寫入 Obsidian，完成了完美替代方案。
